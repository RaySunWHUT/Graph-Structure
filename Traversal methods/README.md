# 图遍历(Graph Traversal)

从图中的任一 **顶点(Vertex)** 出发，对图中的所有顶点访问一次且只访问一次的次序序列。

​	**BFS(Breadth Trist Search)** 类似于 **树(Tree)** 的 **层次遍历**.
